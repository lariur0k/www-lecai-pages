# www-lecai-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，主要面向静态网页的存储与展示。  
页面内容不针对特定域名或网站，仅作为通用 HTML 资源的集中管理仓库。

## 目录说明

```
www-lecai-pages/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的样式、脚本、图片等资源文件
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名即为页面标识。
- `assets/`：按需存放页面所需的静态资源，如 CSS、JS、图像等。

## 页面归档说明

- 所有页面均为纯前端 HTML，不包含服务端逻辑。
- 页面之间相互独立，无依赖关系。
- 归档内容可能随时间更新或清理，不保证长期稳定可用。
- 建议在引用或分发前自行确认页面内容与链接的有效性。
